## What I do
Take a long text and produce a concise summary.

## When to use me
- User pastes a long article or document and asks for a summary
- User says "summarize this" followed by multiple paragraphs
- The input text is genuinely long (200+ words) and needs condensing

## When NOT to use me
- User asks a short question — just answer it
- User sends a few sentences — no need to summarize
- User asks you to explain something — explain it yourself
- User is having a conversation — respond naturally

## Example inputs
- text: (a 500-word article about climate change)

## Example outputs
- A 2-3 sentence summary of the key points
