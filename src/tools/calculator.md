## What I do
Evaluate mathematical expressions and return the result.

## When to use me
- User asks to calculate something with actual numbers
- User says "what is 2 + 3", "compute 15% of 200", "calculate 7 * 8"
- The task requires arithmetic, percentages, or math operations

## When NOT to use me
- User asks a question that happens to contain a number ("how many planets are there?")
- User greets you or makes conversation
- User asks about concepts, history, or anything non-mathematical
- You already know the answer without computing

## Example inputs
- expression: "2 + 3 * 4"
- expression: "100 * 0.15"
- expression: "(50 + 30) / 2"

## Example outputs
- "2 + 3 * 4 = 14"
- "100 * 0.15 = 15"
- "(50 + 30) / 2 = 40"
