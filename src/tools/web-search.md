## What I do
Search the web for current information and return top results.

## When to use me
- User asks about something that requires up-to-date information
- User asks "what is the current price of X" or "latest news about Y"
- You genuinely do not know the answer and need to look it up

## When NOT to use me
- User asks a general knowledge question you can answer directly
- User asks about well-known facts, concepts, or history
- User is making conversation or asking your opinion
- User asks "what is gravity" or "explain photosynthesis" — you know these

## Example inputs
- query: "current weather in Minneapolis"
- query: "latest EU AI Act updates 2026"

## Example outputs
- Summary of search results with relevant snippets
- "No results found" if the query returns nothing
